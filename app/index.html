THE eXCHANGE
THE eXCHANGE is a bible study app that allows the user to search for a bible study by specific search categories. This app also allows the user to search for bible studies nearby or within a certain mile range. Once the user signes up for a bible study group, the user can engage and interact with fellow group members.

Having built the entire back-end of the project, this project helped me to understand the relationship between the front-end and back-end engineering process of an app.

Tools Utilized: HTML CSS SASS ANGULAR BOOTSTRAP HEROKU
